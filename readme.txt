=== Cat Quotes ===

== Description == 
This Plugin is based on the "Hello Dolly" WordPress Plugin , When enabled you will randomly see a Cat Quote, in the upper right of your admin screen.This plugin is currently in early beta, if you would like to test it, download and activate, post any suggestions or errors in the quest book.

== Installation ==  
  a) Unzip the Cat_Quote.zip file , and upload Cat_Quote.php to your wp-content/plugin 
  b) Activate your plugin in your control Panel .

Requires at least : should work with any version above 2.0 , as it is based off the hello dolly plugin

Stable tag: beta

Tested up to: 2.3
